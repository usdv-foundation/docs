| Role       | Functions                                                                                                                                            |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Foundation | <ol><li>Blacklist users from mint/burn/transfer/transferFrom</li></ol>                                                                               |

| Vault      | <ol><li>Mint </li><li>Burn </li></ol>                                                                                                                |
| Owner      | <ol><li>Upgrade Contract</li><li>Set Role</li></ol>                                                                                                  |
